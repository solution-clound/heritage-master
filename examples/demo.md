## 场景五：知识问答

### 对话示例

```
用户：四大名绣是哪些？有什么区别？

助手：让我为您解答四大名绣的相关知识。

[调用 ask_heritage_question(question="四大名绣是哪些？有什么区别？")]

# 四大名绣

四大名绣是指中国刺绣中最具代表性的四种绣品：

## 1. 苏绣（江苏）
- 特点：精细雅洁，针法细密
- 代表作：《猫》、《金鱼》
- 技艺：双面绣、乱针绣

## 2. 湘绣（湖南）
- 特点：形象生动，色彩鲜艳
- 代表作：《狮虎》、《雄鹰》
- 技艺：鬅毛针、双面全异绣

## 3. 粤绣（广东）
- 特点：构图饱满，繁而不乱
- 代表作：《百鸟朝凤》、《九龙屏风》
- 技艺：金银线垫绣、钉金绣

## 4. 蜀绣（四川）
- 特点：针法严谨，平齐光亮
- 代表作：《芙蓉鲤鱼》、《熊猫》
- 技艺：晕针、铺针、滚针

## 主要区别
- 苏绣：精细，适合表现细腻的质感
- 湘绣：写实，擅长表现动物毛发
- 粤绣：华丽，喜用金银线装饰
- 蜀绣：明快，色彩对比强烈

每种绣法都有其独特魅力，值得细细品味。
```

## 场景十：Web API 使用

### Python 示例

```python
import httpx

# 搜索非遗项目
response = httpx.get(
    "http://localhost:8000/api/search",
    params={"q": "刺绣", "region": "江苏"}
)
print(response.json())

# 知识问答
response = httpx.post(
    "http://localhost:8000/api/ask",
    json={"question": "四大名绣是哪些？"}
)
print(response.json()["data"]["answer"])

# 规划旅行
response = httpx.post(
    "http://localhost:8000/api/trip",
    json={"city": "杭州", "days": 2, "interests": ["传统技艺"]}
)
print(response.json())
```

### JavaScript 示例

```javascript
// 搜索非遗项目
const response = await fetch('http://localhost:8000/api/search?q=刺绣&region=江苏');
const data = await response.json();
console.log(data);

// 查找场馆
const venues = await fetch(
  'http://localhost:8000/api/venues?city=北京&keyword=博物馆'
);
console.log(await venues.json());
```

### cURL 示例

```bash
# 搜索
curl "http://localhost:8000/api/search?q=昆曲"

# 详情
curl "http://localhost:8000/api/project/昆曲"

# 场馆
curl "http://localhost:8000/api/venues?city=广州"

# 问答
curl -X POST "http://localhost:8000/api/ask" \
  -H "Content-Type: application/json" \
  -d '{"question": "四大名绣是哪些？"}'
```
